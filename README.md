# Online World Radio 🌍📻

An interactive, high-performance web-based application that maps live audio streams from around the globe onto an intuitive geospatial interface. Users can seamlessly pan, zoom, and select regional nodes to listen to live broadcasts across continents in real-time.

## ✨ Features Visible in UI
* **Interactive Map Discovery:** Features a responsive Leaflet-powered global map cluster system allowing users to explore stations dynamically by geographical location.
* **Live Streaming Infrastructure:** Integrated HTML5 audio playback architecture capable of handling active live streams with varied bitrates and codecs (e.g., AAC+, MP3).
* **Comprehensive Metadata Tagging:** Dynamic station cards organized by country codes (`FR`, `DE`, `US`, `NL`) and live category tags (`news`, `pop`, `culture`, `talk`).
* **Rich Audio Controls:** Sleek media footer equipped with volume normalization, custom play/pause streaming logic, station favoriting, and sleep configurations.
* **Power User Navigation:** Built-in keyboard shortcut commands (`⌘ K`) and specialized layout filters for rapid station discovery.

## 🛠️ Tech Stack
* **Frontend Core:** JavaScript (ES6+), HTML5 Canvas/Audio, CSS3 Custom Properties
* **Mapping Framework:** Leaflet.js, OpenStreetMap, CARTO Tiles
* **Data Layer:** Dynamic JSON-based global station registry (250+ stations)
